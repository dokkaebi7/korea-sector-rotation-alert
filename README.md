# Korea Sector Rotation Alert

한국 주식 시장 섹터 순환매(로테이션)를 감지하고 알림을 주는 툴입니다.  
ETF를 통해 섹터별 수급(거래대금 증가율) + 상대강도(RS) + 뉴스 내러티브를 분석해 매수/관망 신호를 제공합니다.

현재 주요 섹터: 반도체, 바이오, 2차전지, 자동차, 신재생에너지 등  
(2026년 기준 한국 시장 로테이션 테마 중심)

## 주요 기능
- **수급 분석**: 최근 5일 거래대금 평균 vs 이전 5일 비교 (증가율 % 계산)
- **상대강도(RS)**: 섹터 5일 수익률 - 시장(KODEX 200) 수익률
- **내러티브 체크**: 네이버 뉴스 최신 헤드라인 상위 3개 스크래핑
- **정렬 & 상태 표시**: 수급 증가율 기준 내림차순 + 🔥/🟡/💤 이모지 신호
- **데이터 소스**: FinanceDataReader (KRX 기반 안정적 데이터)
- **설정 커스터마이징**: config.yaml 파일로 섹터 목록, 분석 기간, 임계값 등 자유롭게 변경 가능
